# How to Update Board
1. Open up Github Codespaces
2. Open up the assets directory (GW-Temp/src/assets)
3. Add and/or remove pictures (accepts png, jpg, webm)
4. In the terminal type in these commands.
    a. "cd GW-Temp"
    b. "npm run build"
    c. "npm run deploy"
5. In a few minutes the website should update
